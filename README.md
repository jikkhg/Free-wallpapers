## 🎯 Key Features Explained

### Search & Filter
- Type in the search box to find wallpapers by name
- Click category buttons to filter by type
- Both work together seamlessly

### Preview Mode
- Hover over any card and click the eye icon
- Or click the image directly to preview
- Use arrow keys to navigate between wallpapers
- Press ESC to close the preview

### Download
- Click the download button on any card
- Wallpaper will be automatically downloaded to your device

## 🔧 Customization

### Adding New Wallpapers

Add new cards in `index.html`:

```html
<div class="card" data-category="gaming">
  <div class="card-image">
    <img src="your-image-url" alt="Description" loading="lazy">
    <div class="card-overlay">
      <button class="preview-btn">👁️ ดูตัวอย่าง</button>
    </div>
  </div>
  <div class="content">
    <span class="category-tag">GAMING</span>
    <h2>Wallpaper Name</h2>
    <button class="download-btn">⬇️ ดาวน์โหลด</button>
  </div>
</div>
```

### Changing Colors

Edit CSS variables in `style.css`:

```css
:root {
  --primary-color: #0f0f1e;
  --accent-color: #00d4ff;
  /* ... more colors */
}
```
