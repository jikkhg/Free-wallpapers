tl;dr — the short version

เราทำระบบนี้ขึ้นมาเพื่อให้คุณค้นหาไอเดียภาพสวยๆ ได้ง่ายขึ้น คุณเอาภาพไปใช้ส่วนตัวได้ฟรี แต่ถ้าจะเอาไปใช้ขายของหรือทำธุรกิจ โปรดเช็คลิขสิทธิ์กับเจ้าของภาพต้นฉบับ (Pexels) อีกครั้งเพื่อความชัวร์นะ!

⚠️ Remark: ข้อมูลภาพถูกดึงผ่าน API เพื่อการอ้างอิงเท่านั้น Pika Studio ไม่ได้เป็นเจ้าของลิขสิทธิ์ภาพเหล่านี้

1. การยอมรับข้อกำหนด

การเข้าถึงหรือใช้งานเว็บไซต์ PikaSearch ถือว่าคุณได้อ่าน เข้าใจ และตกลงที่จะผูกพันตามข้อกำหนดและเงื่อนไขเหล่านี้ทั้งหมด หากคุณไม่ตกลงตามข้อกำหนดใดๆ โปรดหยุดการใช้งานเว็บไซต์ทันที

2. สิทธิ์ในทรัพย์สินทางปัญญา

รูปภาพทั้งหมดที่ปรากฏบนเว็บไซต์นี้ดึงข้อมูลมาจากแหล่งภายนอกผ่าน API (เช่น Pexels) โดยที่:

  • รูปภาพเป็นลิขสิทธิ์ของเจ้าของภาพหรือช่างภาพต้นฉบับ
  • PikaSearch เป็นเพียงผู้ให้บริการระบบค้นหาและแสดงผลภาพเท่านั้น
  • ผู้ใช้ต้องตรวจสอบสิทธิ์การใช้งาน (License) ของแต่ละรูปภาพก่อนนำไปใช้ในเชิงพาณิชย์
  • ห้ามมิให้ผู้ใดคัดลอกส่วนประกอบของเว็บไซต์ (Code, UI Design, Logos) ไปใช้โดยไม่ได้รับอนุญาต

3. นโยบายความเป็นส่วนตัว (Privacy Policy)

ความเป็นส่วนตัวของคุณสำคัญสำหรับเรา เราจัดเก็บข้อมูลเพียงเล็กน้อยเพื่อให้ระบบทำงานได้:

  • ข้อมูลการค้นหา: เราอาจบันทึกคำค้นหาเพื่อปรับปรุงอัลกอริทึมการแสดงผล
  • LocalStorage: เราใช้ LocalStorage บนเบราว์เซอร์ของคุณเพื่อเก็บ "รายการโปรด" (Favorites) ซึ่งข้อมูลนี้จะอยู่บนเครื่องของคุณเท่านั้น เราไม่ได้เก็บไว้บนเซิร์ฟเวอร์ของเรา
  • IP Address: อาจมีการบันทึกเพื่อป้องกันการโจมแดทางไซเบอร์และตรวจสอบสถิติการเข้าใช้งาน

4. ข้อกำหนดเรื่องอายุ

เว็บไซต์นี้เหมาะสำหรับผู้ที่มีอายุ 13 ปีขึ้นไป หากคุณมีอายุน้อยกว่า 13 ปี คุณควรใช้งานภายใต้การดูแลของผู้ปกครอง

5. นโยบายโฆษณาและคุกกี้

เรามีการใช้บริการโฆษณาจากบุคคลที่สาม (เช่น Google AdSense) ซึ่งอาจมีการใช้คุกกี้เพื่อเก็บข้อมูลการเยี่ยมชมเว็บไซต์ของคุณ:

  • คุกกี้ถูกใช้เพื่อแสดงโฆษณาที่ตรงกับความสนใจของคุณ
  • ข้อมูลที่เก็บรวบรวมไม่ระบุตัวตนและใช้เพื่อการวิเคราะห์สถิติเท่านั้น
  • คุณสามารถเลือกปิดการใช้งานคุกกี้ได้ผ่านการตั้งค่าในเบราว์เซอร์ของคุณ

6. การปฏิเสธความรับผิดชอบ (Disclaimer)

PikaSearch ไม่รับประกันว่าบริการจะไม่ถูกขัดจังหวะ หรือไม่มีข้อผิดพลาด เราจะไม่รับผิดชอบต่อความเสียหายใดๆ ที่เกิดขึ้นจากการใช้งานเว็บไซต์ ไม่ว่าทางตรงหรือทางอ้อม

ข้อมูลภาพและลิงก์ภายนอกถูกจัดทำขึ้นเพื่อความสะดวกเท่านั้น เราไม่สามารถควบคุมเนื้อหาบนเว็บไซต์ภายนอกเหล่านั้นได้

7. การแก้ไขเปลี่ยนแปลง

เราขอสงวนสิทธิ์ในการแก้ไขหรือเปลี่ยนแปลงข้อกำหนดเหล่านี้ได้ตลอดเวลาโดยไม่ต้องแจ้งให้ทราบล่วงหน้า การที่คุณใช้งานเว็บไซต์ต่อไปหลังจากมีการเปลี่ยนแปลงถือว่าคุณยอมรับข้อกำหนดที่แก้ไขแล้ว
